TextDB

November 2014

TextDB was a hackathon project that my friends and I worked on at Wildhacks 2014. The idea behind the project was to create a way to store data on the fly by configuring databases using 7 meta data tables using PostgreSQL. TextDB is a flask application that is also integrated with the Twilio API to allow users to text data via sms to their database keys. Users can then access this information online.

Technology Used: Python, Flask, PostgreSQL, SQLAlchemy, Boostrap
