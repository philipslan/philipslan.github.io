Work

Atlasify

June 2014 - Present

Atlasify is a new way to visualize how different concepts are related: maps, periodic tables, and other reference systems. On the Atlasify project I worked on creating explanations between two search queries. My partner and I created a system that took two search concepts from a wikipedia api that Professor Downey had previously developed, and took small snippets of text from the top ranked shared inlinks. Initially this project was written in as a php app but we have since ported over to node.js environment.

Technology Used: Node.js, python, mySQL, php, jade, express

TextDB

November 2014

TextDB was a hackathon project that my friends and I worked on at Wildhacks 2014. The idea behind the project was to create a way to store data on the fly by configuring databases using 7 meta data tables using PostgreSQL. TextDB is a flask application that is also integrated with the Twilio API to allow users to text data via sms to their database keys. Users can then access this information online.

Technology Used: Python, Flask, PostgreSQL, SQLAlchemy, Boostrap
